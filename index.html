Executive Summary

ESE (EPROM Smart Engineering) is an AI-powered engineering copilot for petroleum process engineers. Built by EPROM for EGYPS 2026, it now needs a clear business model to move from conference demo to revenue-generating product.

Egypt's economic reality — EGP devaluation, tight refinery budgets, and procurement constraints — demands pricing that EGPC subsidiaries can actually approve. This model reflects that ground truth.

Two-Track Strategy: Model A (Individual SaaS) generates cash flow and user feedback. Model B (Corporate SaaS) lands ESE inside 10 EGPC refineries at honest Egyptian pricing. Together they build the reference base for higher-ticket Models C/D in Years 3-5.

Combined Strategy — How They Fit Together

Model A feeds Model B

User feedback from 1,500+ engineers tells us exactly what refiners need. Case studies from Pro users arm EGPC pitches. Cash flow from subscriptions helps carry shared costs.

Model B builds for Model C

10 EGPC refineries as references = the credential to sell Model C on-prem appliances at 10× the ticket size. Customer success stories, KPIs, and relationships all compound.

Egyptian sovereignty

Data stays in Egypt. AI built by Egyptian engineers. Priced for Egyptian budgets. No foreign API dependency for operational data. Law 151/2020 compliant by design.

Approval gates

1) 3 pilots signed Y1Q4. 2) v2 product ready Y1Q3. 3) Model A running in parallel. If gates missed → revert to Model A-only.
